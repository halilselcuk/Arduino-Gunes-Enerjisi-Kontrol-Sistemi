# Arduino Güneş Enerjisi Kontrol Sistemi

## Kurulum

 - SD klasöründeki dosyaları W5100'e takılacak olan SD karta yükleyin.
 - Gerekli Arduino kütüphanelerini yükleyin.
 - Arduino kodlarında 10. satırdaki IP adresini ayarlayın.
